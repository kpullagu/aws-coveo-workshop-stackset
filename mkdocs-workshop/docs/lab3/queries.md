# Lab 3 Sample Queries

Use these query sets to validate the native Coveo Search Agent flow.

## ACH Basics

Initial question:

```text
What is ACH and when is it commonly used?
```

Follow-ups:

```text
How is that different from a wire transfer?
```

```text
Which one is better for recurring payments?
```

Expected observations:

- first answer is grounded and cited
- follow-ups understand the prior topic
- latest answer stays expanded

## Credit Report Errors

Initial question:

```text
I found errors in my credit report. What should I do first?
```

Follow-ups:

```text
What timing should I keep in mind?
```


Expected observations:

- follow-ups do not require restating "credit report errors"
